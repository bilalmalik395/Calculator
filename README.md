# Professional Animated Calculator

A modern **Animated Calculator Web App** built using **HTML, CSS, and JavaScript**.  
This project features a clean glassmorphism UI, animated background, and a calculation history system.

## 🚀 Preview
A responsive and interactive calculator with smooth animations and a professional interface.

## 🛠 Technologies Used
- **HTML5** – Structure of the calculator
- **CSS3** – Styling, animations, glassmorphism design
- **JavaScript** – Calculator logic and functionality

## ✨ Features
- Animated gradient background
- Modern glassmorphism UI design
- Basic calculator operations (+, −, ×, ÷, %, decimals)
- Keyboard input support
- Calculation history table
- Clear display and clear history option
- Smooth button hover effects
- Responsive layout

## ⚙️ How It Works
The calculator interface is built with **HTML** and styled using **CSS animations and glassmorphism effects**.  
**JavaScript** handles user input, performs calculations, and dynamically updates the display.

When a user enters an expression and presses **=**, the result is calculated and displayed.  
Each calculation is also saved in the **history table**, allowing users to review previous expressions and results.
